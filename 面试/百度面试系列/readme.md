# 3. 输出题
const obj1 = {a:1}
const obj2 = {a:2}
console.log(obj1 == obj2)
console.log(obj1 === obj2)


首先两个等号只会判断值相等,发生隐式转化
会发生字符串,但对象的判断规则当中会有判断引用地址

三个等号是严格相等,不发生类型转换,通过比较类型是否相同,值是否相同
